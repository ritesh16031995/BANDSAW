# BANDSAW MACHINE AUTOMATION
This project demonstrates the complete PLC and HMI development for an industrial Bandsaw Machine. The automation system was designed to control machine operation in both Automatic and Manual modes while providing a user-friendly HMI interface for monitoring, operation, and troubleshooting.
The project includes PLC programming, HMI screen development, machine sequencing, encoder-based control, operator controls, and machine status monitoring.

## Project Objectives
* Automate the bandsaw cutting process.
* Provide Manual and Automatic operating modes.
* Improve operator interaction through HMI.
* Monitor machine inputs and outputs in real time.
* Implement safe machine operation and interlocks.
* Control machine movement using encoder feedback.

## Machine Functions

### Automatic Mode
In Automatic Mode, the machine performs the complete cutting sequence automatically based on programmed logic and machine feedback.
* Automatic machine cycle
* Position monitoring
* Encoder-based movement control
* Sequence control
* Output interlocking
* Cycle completion indication

### Manual Mode
Manual Mode allows maintenance personnel and operators to individually operate machine components for testing and troubleshooting.
* Manual actuator operation
* Jog controls
* Maintenance testing
* Sensor verification
